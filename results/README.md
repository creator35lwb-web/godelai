## How to Contribute Test Results

1. **Run a test** using either:
   - Local Python: `python tests/test_mirror.py`
   - Google Colab: Use the verification notebook

2. **Save your results**:
   - Notebooks: `.ipynb` files
   - Logs: `.txt` or `.md` files
   - Screenshots: `.png` files

3. **Name your file** with the format:
   ```
   GODELAI_<TestType>_<Number>_by_<YourName>.<ext>
   ```

4. **Submit a PR** or upload directly if you have access
